# AI Concepts Assistant – Mini LLM Chatbot

This project is a mini language model built using PyTorch that answers basic AI-related questions.

# Features
- Custom AI dataset
- Character-level tokenizer
- Text generation
- Trained using PyTorch

# Tech Stack
- Python
- PyTorch
- Google Colab

# How It Works
User Input → Tokenization → Model → Response Generation

# Limitations
- Small dataset
- Basic responses
